# 📚 나만의 복습 퀴즈 앱 (Review Quiz App)

HTML, CSS, Vanilla JavaScript를 활용하여 만든 간단하고 직관적인 **웹 기반 퀴즈 애플리케이션**입니다. 
학습한 내용(예: Java 기초 개념)을 스스로 점검하고 복습하기 위한 목적으로 제작되었습니다.

## 🌐 라이브 데모 (Live Demo)

아래 링크를 클릭하면 별도의 설치 없이 브라우저에서 바로 퀴즈를 풀어볼 수 있습니다.

## 💡 주요 기능

* **동적 퀴즈 렌더링**: JavaScript 배열 객체(`quizData`)에 저장된 질문과 보기를 화면에 자동으로 불러옵니다.
* **즉각적인 피드백**: 
    * 사용자가 답을 선택하면 즉시 정답/오답 여부를 시각적으로 제공합니다.
    * 정답을 맞히면 초록색, 틀리면 빨간색으로 버튼 색상이 변경되며 원래 정답을 함께 표시해 줍니다.
* **중복 선택 방지**: 답안을 한 번 선택하면 다른 버튼들이 비활성화되어 중복 입력을 막습니다.
* **결과 집계 및 재도전**: 
    * 모든 문제를 풀고 나면 최종 맞힌 문제 수와 점수를 결과 화면에 출력합니다.
    * `다시 풀기` 버튼을 통해 초기화 후 재도전할 수 있습니다.

## 🛠 기술 스택

* **Markup**: HTML5
* **Styling**: CSS3 (Flexbox를 활용한 중앙 정렬 및 반응형 UI 디자인)
* **Logic**: Vanilla JavaScript (ES6+)
